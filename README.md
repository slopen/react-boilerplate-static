# react-boilerplate-static

* react
* react-router
* reactstrap
* express


### install

```
$ git clone git@github.com:slopen/react-boilerplate-static.git
$ cd react-boilerplate-static
$ yarn
```

### production

```
$ yarn run production
```

* open `http://localhost:9020`

### development

```
$ yarn run dev
```

* open `http://localhost:9021`
